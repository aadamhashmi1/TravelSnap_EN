# TravelSnap

**Branch:** `lesson-2`

## 🎯 Practical Task: Lesson 2 — JSX, Components, and Props

### Core Task

1. Create a `TripCard` component with the following props:
   - `title: string`
   - `destination: string`
   - `date: string`
   - `rating: number`
2. Display 3–5 static travel cards on the home screen inside a `ScrollView`.
3. Use TypeScript interfaces to type the props.

### Extension ★

**Create a `RatingStars` component:**
- Based on the `rating` prop (1–5), display the appropriate number of stars (e.g., using ★ and ☆ emojis).
- Use this component inside the `TripCard`.
